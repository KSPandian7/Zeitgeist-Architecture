03

12-Step Workflow

RSS → Live Instagram
STEP 01
Cron fires every 6h
Parallel fetch across all 25 RSS endpoints. 15s timeout per source. Failures logged silently.
STEP 02
SHA-256 dedup
Title hashed. Existing hash = skip silently. Prevents same story from 5 sources appearing twice.
STEP 03
48h cache clear
Articles older than 48h auto-deleted. Feed always stays fresh and current.
STEP 04
NLP topic + India score
Keyword scoring assigns topic. India-relevance boosts ranking. Trending flag if 3+ sources match.
STEP 05
Caption generated
Topic template fills caption with hook, context, CTA. Hashtags assigned per topic pool.
STEP 06
node-canvas renders
1080×1080 JPG generated server-side. Category pill, title box, branding — pixel perfect every time.
STEP 07
Cloudinary upload
Image uploaded to Cloudinary free CDN. Returns public HTTPS URL for Instagram API.
STEP 08
Review queue
Post appears in dashboard with preview, caption, hashtags. Status = pending.
STEP 09
Human approves
One-click approve. Edit caption inline if needed. Reject removes permanently.
STEP 10
IG media container
Graph API creates container with image URL + caption. Returns creation_id.
STEP 11
Publish to Instagram
2s wait → media_publish call → post goes live. IG post ID logged in database.
STEP 12
History logged
Post status → posted. IG post ID saved. Timestamp recorded. Full audit trail.
